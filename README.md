# dotfiles

Integrated terminal configuration.

![UIE](./screenshot.png)

## How to install

```sh
git clone git@github.com:ucchiee/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow */
```

You can also install useful tools.

```sh
pip install ranger-fm
```

## Useful Tools

- alacritty
- asdf
- ncdu
- neovim
- ranger
- tmux
